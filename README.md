## 🌌 The Vision

Modern productivity tools see you as a list of tasks to complete. **SurjyoPath sees you as a universe to explore.**

Instead of disconnected notes, checklists, and metrics, your life becomes a **living solar system** — with **you** at the center as the Sun, and every aspect of your growth orbiting around you: your thoughts, your goals, your knowledge, your creativity, your relationships, and your identity.

SurjyoPath is not another productivity app. It is a **personal growth operating system** — a place where technology doesn't replace your thinking, but *illuminates* it.

## ✨ The Planets — Core Features

Each **planet** in your solar system represents a fundamental dimension of personal growth:

<br />

### 🌞 **Galaxy Dashboard** — *Your Cosmic Home*

The entry point to your universe. A stunning, interactive solar system where the Sun — ***you*** — sits at the center with pulsating corona rings, while six planets orbit gracefully in elliptical paths. Hover to explore, click to journey.

- Animated orbital mechanics with fluid rotation
- Pulsating central sun with layered glow effects
- Interactive planets with hover states and labels
- Ambient starfield background
- Smooth transitions between modules

<br />

### 📖 **AI-Powered Journal** — *The Knock Engine*

Every thought you record is met not with silence, but with intelligence. The **Knock AI** reads between the lines — uncovering hidden patterns, surfacing blind spots, and asking the questions you didn't know to ask yourself.

- ✍️ Rich thought composition with titles and categories
- 🤖 **Knock Engine** — analyzes each entry for emotional patterns, cognitive biases, and reflective opportunities
- 🏷️ Auto-tagging and categorization
- 📊 Visual mood and thought pattern timelines
- 🔒 Private by default, shareable when you choose

> "What you wrote about resilience three times this month… what's that telling you?"

<br />

### 🎯 **Adaptive Learning Pathways** — *The Pathway Engine*

Turn any ambition into a structured learning journey. Tell SurjyoPath "I want to learn quantum computing" or "I want to become a better storyteller" — and the **Pathway AI** generates a complete, personalized curriculum.

- 🗺️ Auto-generated course structures with milestones
- 📚 Chapter-by-chapter learning progression
- ✅ Built-in assessments and quizzes
- 🎯 Progress tracking with completion metrics
- 🔄 Dynamic adaptation based on your pace

<br />

### 📚 **Integrated Digital Library**

A beautiful reading companion that respects the depth of your learning. Upload books or articles and track your progress at the chapter level.

- 📖 Chapter-level reading progress
- 🏷️ Tagging and categorization
- 📊 Reading statistics and habits
- 🌓 Beautiful reading interface with dark/light modes
- 🔗 Direct integration with goals and publications

<br />

### ✍️ **AI-Assisted Publications** — *The Polish Engine*

Your voice matters. The **Polish AI** refines your drafts into publication-ready content — without erasing *you*. It preserves your unique tone, style, and personality while improving clarity and flow.

- 🖋️ Write with the confidence that your voice is protected
- 🤖 **Polish Engine** — smart suggestions, grammar improvements, structural feedback
- 📰 Public articles with community engagement
- 💬 Comments, likes, and sharing
- 🌐 Public profiles for published authors

<br />

### 👥 **Social Community — Friends & Connections**

Growth is better together. Connect with others on their own journeys, share insights, and build a network of mutual growth.

- 👤 Friend profiles with personal journals and publications
- ❤️ Like and comment on publications

- 🛡️ Privacy controls for every piece of content

## 🤖 The Three AI Engines

SurjyoPath features **three specialized AI engines**, each designed for a distinct purpose. Together, they form an intelligent companion that *augments* your thinking rather than replacing it.

```
                    ┌──────────────────────┐
                    │   ☀️  SurjyoPath      │
                    │   (You — The Center)  │
                    └──────────┬───────────┘
                               │
            ┌──────────────────┼──────────────────┐
            │                  │                  │
            ▼                  ▼                  ▼
    ┌──────────────┐  ┌──────────────┐  ┌──────────────┐
    │  KNOCK       │  │  PATHWAY     │  │  POLISH      │
    │  Engine      │  │  Engine      │  │  Engine      │
    ├──────────────┤  ├──────────────┤  ├──────────────┤
    │ Journal      │  │ Goals        │  │ Publications │
    │ Analysis     │  │ Curriculum   │  │ Refinement   │
    │ Patterns     │  │ Generation   │  │ Voice        │
    │ Blind Spots  │  │ Milestones   │  │ Preservation │
    │ Reflection   │  │ Assessments  │  │ Clarity      │
    └──────────────┘  └──────────────┘  └──────────────┘
```

All three engines run on a **Supabase Edge Function** powered by **MiniMax M2.7** through **Fireworks AI**, accelerated by **AMD MI300X GPUs**, with a graceful browser-based fallback.

## 🛠️ Technology Stack

| Layer | Technology | Why |
|-------|-----------|-----|
| **Frontend** | React 19 + TypeScript 5.8 | Type safety, bleeding-edge reactivity |
| **Styling** | Tailwind CSS v4 | Utility-first, design tokens, OKLCH color space |
| **Build** | Vite 6 | Lightning-fast HMR and builds |

| **Routing** | React Router 7 | Declarative, nested layouts, auth guards |
| **Backend** | Supabase (Postgres + Auth + Edge Functions) | Open-source Firebase alternative |
| **AI Inference** | Fireworks AI + MiniMax M2.7 on AMD MI300X | Fast, scalable, intelligent |
| **Icons** | Lucide + React Icons | Consistent, modern iconography |
| **Auth** | Supabase Auth (Email/Password) | Secure, JWT-based, session management |

## 🚀 Getting Started

### Prerequisites

- Node.js 20+
- npm 10+
- A Supabase project (or use mock mode)

### Installation

```bash
# Clone the repository
git clone https://github.com/surjyopath/surjyopath.git
cd surjyopath

# Install dependencies
npm install

# Start the development server
npm run dev
```

Visit `http://localhost:5173` — and watch your universe come to life. 🪐

### Configuration

Create a `.env` file based on the example:

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
VITE_GO_BACKEND_URL=http://localhost:8080  # Optional: Go backend
```

> **No API keys needed for development** — SurjyoPath gracefully falls back to browser-based mock AI when the edge function is unavailable.

### Build for Production

```bash
npm run build
npm run preview
```

## 🔐 Authentication & Privacy

SurjyoPath uses **Supabase Auth** with JWT-based sessions. All journal entries are private by default. Publications can be shared publicly at the user's discretion.

- **Guest mode** — explore the galaxy without signing up
- **Email/password auth** — secure, familiar, reliable
- **Protected routes** — your data stays yours
- **Rate limiting** — AI calls are throttled to ensure fair usage

## 🧡 The Story Behind the Name

**Surjyo** — from the Sanskrit word **सूर्य (Sūrya)**, meaning *Sun*.
**Path** — the journey you walk.

**SurjyoPath** = *The Sun's Way*.

Because every journey of growth begins with the light you carry within you. You are not a passenger in your life — you are the star at the center of your own universe.
