## 🤖 The Three AI Engines

SurjyoPath features **three specialized AI engines**, each designed for a distinct purpose. Together, they form an intelligent companion that *augments* your thinking rather than replacing it.

```
                    ┌──────────────────────┐
                    │   ☀️  SurjyoPath      │
                    │   (You — The Center)  │
                    └──────────┬───────────┘
                               │
            ┌──────────────────┼──────────────────┐
            │                  │                  │
            ▼                  ▼                  ▼
    ┌──────────────┐  ┌──────────────┐  ┌──────────────┐
    │  KNOCK       │  │  PATHWAY     │  │  POLISH      │
    │  Engine      │  │  Engine      │  │  Engine      │
    ├──────────────┤  ├──────────────┤  ├──────────────┤
    │ Journal      │  │ Goals        │  │ Publications │
    │ Analysis     │  │ Curriculum   │  │ Refinement   │
    │ Patterns     │  │ Generation   │  │ Voice        │
    │ Blind Spots  │  │ Milestones   │  │ Preservation │
    │ Reflection   │  │ Assessments  │  │ Clarity      │
    └──────────────┘  └──────────────┘  └──────────────┘
```

All three engines run on a **Supabase Edge Function** powered by **MiniMax M2.7** through **Fireworks AI**, accelerated by **AMD MI300X GPUs**, with a graceful browser-based fallback.

## 🚀 Getting Started

### Prerequisites

- Node.js 20+
- npm 10+
- A Supabase project (or use mock mode)

### Installation

```bash
# Clone the repository
git clone https://github.com/surjyopath/surjyopath.git
cd surjyopath

# Install dependencies
npm install

# Start the development server
npm run dev
```

Visit `http://localhost:5173` — and watch your universe come to life. 🪐

### Configuration

Create a `.env` file based on the example:

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
VITE_GO_BACKEND_URL=http://localhost:8080  # Optional: Go backend
```

> **No API keys needed for development** — SurjyoPath gracefully falls back to browser-based mock AI when the edge function is unavailable.

### Build for Production

```bash
npm run build
npm run preview
```

## 🔐 Authentication & Privacy

SurjyoPath uses **Supabase Auth** with JWT-based sessions. All journal entries are private by default. Publications can be shared publicly at the user's discretion.

- **Guest mode** — explore the galaxy without signing up
- **Email/password auth** — secure, familiar, reliable
- **Protected routes** — your data stays yours
- **Rate limiting** — AI calls are throttled to ensure fair usage
